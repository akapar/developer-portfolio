GAZELIST

Hierarchical task management combined with task-bound focus timers and zero-compromise cloud sync.

Section 01 // Project Architecture

Task Hierarchy & Folder Organization

Break away from flat, unstructured checklists. GAZELIST enables you to model your life, work, and creative ideas exactly as they exist: in rich, nested hierarchies.

Nested Subtasks

Subdivide complex goals into micro-milestones. Create sub-tasks inside high-level folders, or nest tasks under other tasks to any depth without limits.

Drag-and-Drop Nesting

Re-organize list architecture with kinetic drag gestures. Drop a task onto another to nest it, or slide it out to elevate its hierarchical rank.

Smart Folders (ALL CAPS)

Create folders instantly. Typing a task name in ALL CAPS (e.g., "WORK" or "HEALTH") immediately converts the element into a folder.

Folder Controls

Collapse or expand nodes with a single tap. Folders display a subtask counter (e.g., 5 tasks) and offer quick sorting by due dates.

Section 02 // Deep Work Module

Integrated Focus Timer & Session Control

Bridge the gap between task management and active execution. Link your Pomodoro focus intervals directly to individual tasks in your list.

Task-Bound Timers

Start Pomodoro timers directly from any task tile. The session remains tied to the task, logging real-time focus analytics automatically.

Interval Management

Maintain peak performance. Enjoy smooth, customizable transitions between focus intervals, short breaks, and long breaks.

Custom Notification Audio

High-quality chimes to end sessions: Tingsha Cymbal (default), Glass Cup Spoon Chime, or Bass & Drums Loop (130 BPM ambient track).

Audio Settings Playground

Preview and test notification audio, adjust volume settings, or reset back to default parameters with a single tap in the settings panel.

Section 03 // Workspace Metadata

Inline Tags & Smart Metadata

Categorize and filter your task database on the fly using standard markdown formatting right in the text title input.

Instant Tagging

Type # or @ (e.g., #writing, @urgent) inside your title. The parser styles tags in bold automatically for quick visual scanning.

One-Tap Tag Filtering

Tap any tag directly inside a task's title to immediately filter the workspace. Only items carrying that specific tag will remain on screen.

Auto-Complete Panel

Typing # or @ pops up a clean autocomplete overlay above the keyboard, listing active tags sorted by popularity.

Global Tag Directory

The system actively tracks, registers, and indexes your entire database of tags based on usage frequency.

Section 04 // Time Management

Time Planning & Due Dates

Plan out your days, keep an eye on upcoming project milestones, and ensure deadlines are met with natural language scheduling.

Calendar Pickers

Set absolute deadlines using a clean, custom calendar overlay date-picker designed for fast interaction.

Relative Date Indicators

Due dates display in natural language, automatically calculating labels like "Today", "Yesterday", or "3 days later".

Overdue Warnings

Late items are automatically highlighted in red warning tones and sorted under a distinct, high-priority "LATE" header.

Default Due Dates

Toggle an option in settings to automatically schedule all newly created tasks for "Today", eliminating scheduling friction.

Section 05 // Routine Automation

Habit Building & Recurrence

Establish persistent daily routines and track repetitive work obligations with minimal configuration overhead.

Flexible Repeat Rules

Write rules directly to schedule recurring tasks: daily "1d", weekly "1w", or customized interval spans like "~4d".

Automatic Rescheduling

Upon completing a repeating task, the database automatically creates the next occurrence, keeping habit progress uninterrupted.

Section 06 // Interaction Mechanics

Frictionless Action Controls & Editing

Manage lists at lightning speed with advanced touch gestures, batch editors, and inline state changes.

Multi-Select & Batch Actions

Tap task circles to select multiple items. Batch-edit notes, due dates, planned durations, recurrence rules, or perform bulk deletion.

Swipe Gestures

Customize swipe actions. Swipe right to toggle completion, and swipe left to trigger delete, edit, timer, or postponement.

Contextual Popup Menus

Quick access to secondary operations on task tiles: create child items, adjust detail dialogs, add notes, or view category metrics.

Inline Title Editing

Tap any task name text to edit it inline. Pressing Enter or clicking outside instantly saves the changes.

Section 07 // Data & Analytics

Historical Analytics & Reports

Reflect on your accomplishments and discover temporal patterns in your productivity logs with advanced analytics.

Progress Charts

Visualize completed tasks, target milestones, and focused durations over the last 30 days via clean, custom responsive bar charts.

Custom Date Spans

Filter focus logs, completed items, and performance trends using a custom date range picker in the analytics dashboard.

Daily Activity Logs

Inspect detailed historical records of completed tasks grouped by day, displaying exactly how much focus time was spent on each item.

No Lost Work Logs

Even if a task is deleted, its logged focus hours are preserved in reports, styled with a line-through and a (Deleted) label.

Section 08 // Sync & Security

Google Drive Cloud Sync & Security

Sync tasks across all devices automatically without putting your private data on proprietary third-party servers.

100% User-Owned Security Model

Your task databases and focus logs sync directly to your personal Google Drive App Data folder. No mid-tier proxy servers, no database mining, and no tracking. All conflict resolution merges (using local-vs-cloud updatedAt timestamps) happen securely client-side.

UI Accent Themes
